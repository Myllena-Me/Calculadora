
# Calculadora Python 

Este projeto implementa uma calculadora de operações básicas em Python usando uma interface de linha de comando.

# Funcionalidades
- Soma, Subtração, Multiplicação e Divisão
- Tratamento de erros (entrada inválida e divisão por zero)
- Interface intuitiva e simples
